Task Advance

Overview

Features
  • Set production info and machine capacity from sale order line.
  • Create machine and machine capacity.
  • Task of the production project can be group by machine capacity.
  • Split Task in number of tasks.

Add a product in sale order line.

In Production Info section, set the Machine and Production Lead time.

You can create machine capacity also from here.

Production project task dashboard will group by the tasks according to their production project and machine capacity.

Click on the Split Task button will open the new wizard to split a task into number of task.

Enter the number you want to create a task into number of task and click Create task button to create split task.

Enter the production time in the created task and save it.

It will generate the n number of splited task.
